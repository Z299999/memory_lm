# World Files

当前 `world/` 目录里的文件，最好理解成“同一个大世界的不同完整度切片”，而不是四个彼此独立的宇宙。

## 综合完整版

- [`confluence_world.md`](./confluence_world.md)
  - Confluence World
  - 这是当前最完整的综合 world
  - 把人物关系、边疆自然法则、材料反应与转运任务链放进同一个世界里
  - 适合做“某人带某物，经某路，途中遭遇 / 采集 / 改道后，是否还能安全交接”的题型

## 三个局部切面

- [`asterion_lab.md`](./asterion_lab.md)
  - Asterion Lab
  - 偏材料、容器、环境条件、反应与延迟阶段
- [`court_of_veils_world.md`](./court_of_veils_world.md)
  - Court of Veils
  - 偏人物关系、礼制、誓约、谣言热态与政治风险
- [`orthfall_frontier_world.md`](./orthfall_frontier_world.md)
  - Orthfall Frontier
  - 偏高维荒野生存、地貌、补给、异常压力与路径反转

如果只想测试单域学习，可以继续使用三个局部切面。  
如果想测试真正的跨域记忆压缩，优先使用 [`confluence_world.md`](./confluence_world.md)。
