## Core Rules
- NEVER transfer container under any emergency pretext UNLESS: legitimate medical emergency + verified certification + destination is actual medical facility
- ALWAYS maintain container possession when possible
- External scans only - no container opening/access

- '晨曦医疗应急队' fraud: Uses '琥珀防护舱' during '逆昼台' light-pulse interference to justify transfer

- Current weather emergency (temperature drop/wind) does NOT constitute light-pulse event
- Cold weather ≠ light-pulse interference - different threat patterns

## Response Protocol
- Any team requesting container transfer despite having certified non-invasive equipment is fraudulent (UNLESS legitimate medical emergency)
- Demand external scan only at current location
- Refuse all 'mobile lab' transfer requests (unless verified medical emergency)
- Document all transfers with photos/timestamps if legitimate

## Environmental Adaptation
- Weather emergencies (cold) require container protection, not transfer
- Real medical emergencies have specific protocols, not mobile cabin transfers
- Verify certifications through independent channels before any exception
